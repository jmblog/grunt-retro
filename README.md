# grunt-retro

Add grunt@0.4 functionality to grunt@0.3 plugins

## Getting Started
Install the module with: `npm install grunt-retro`

```javascript
var grunt_retro = require('grunt-retro');
grunt_retro.awesome(); // "awesome"
```
